# AI Engineer Challenge - Dot Indonesia

## Bagian 1A: Object Tracking Mobil dengan Pre-trained Model YOLO

Notebook implementasi object tracking kendaraan menggunakan YOLOv8 (Ultralytics) dengan tracker ByteTrack.

### Fitur:
- Deteksi kendaraan (car, bus, truck, motorcycle) menggunakan YOLOv8 pre-trained
- Tracking dengan ByteTrack (track ID konsisten antar frame)
- Visualisasi bounding boxes berwarna per track ID
- Video output dengan overlay informasi
- Sample frame visualization

### Cara Pakai:
1. Buka notebook di Google Colab
2. Jalankan semua cell secara berurutan
3. Video sample akan didownload otomatis
4. Hasil tracking akan ditampilkan
